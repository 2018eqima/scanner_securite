# EQIMA Security Scanner

Plateforme de scan de sécurité EQIMA.

## Branches

| Branche | Rôle |
|---------|------|
| `main` | Production |
| `develop` | Développement actif |

## Stack

- **Backend** : Spring Boot 3.3 / Java 21 — `api-scanner.eqima.org`
- **Frontend** : React / Vite / Tailwind — `scanner.eqima.org`
- **Scanner** : OWASP ZAP — `security.eqima.org`
- **Auth** : Keycloak — `auth.eqima.org`
